## 목적

electron + react를 공부하려고 만든 프로젝트.
별 내용이 없지만 여러 컴퓨터에서 개발하려다 보니까 git에 올리게 되어버렸습니다.

dependency 관련해서는
### `package.json`
파일을 참고해주세요.

초기 셋팅에 관해서는
[Doocong](http://doocong.com/react/react-electron-setup-1/)페이지를 참고했습니다.

버튼이나 입력 폼 디자인은
유튜브 채널 [DarkCode](https://youtu.be/MLfAW55_4cY)을 참고했습니다.

Electron-Builder의 설정에 관해서는
[Suwoni-Codelab](https://suwoni-codelab.com/electron/2017/04/17/Electron-distribution/)페이지를 참고했습니다.

테스트용 스크립트:
### `npm start`

빌드용 스크립트:

윈도우용 빌드 스크립트:
### `npm run dist-win`

리눅스용 빌드 스크립트:
### `npm run dist-linux`

맥용 빌드 스크립트:
### `npm run dist-osx`
맥용은 실제로 안 돌려봐서 동작할지 어떨지 확인이 안 됐습니다.(본인이 맥을 가지고 있지 않기 때문입니다.)
